About

Created for CS362 Project

Save your family and friends the need to carry bulky key chains and the hassle to dig for keys, Fingerprint Garage Opener increases security level of your homes since you no longer need to provide spare keys to housekeepers, or family members. You can simply program their fingerprints into the sensor and erase them whenever you need.

  • Utilizes the latest accurate and fast recognition live layer optical sensor technology

  • 200 unique fingerprint users

  • Easy programming, add or delete user is performed directly on the pad

Our Story

For our 362 Project, we have built a fingerprint garage door opener. Using Kuman UNO R3 as a micro-controller, we designed a system with the capability of mounting a fingerprint scanner on a garage model. This system, upon the placement of a authorized fingerprint, will cause the garage panel to open and close. This project contains highlights of what we learned this semester by incorporating pushbuttons, an LCD screen and micro-controller communication, while exploring new realms with the Fingerprint Scanner Module and stepper motors. With the knowledge of circuit design, we planned to create a system that can be applicable in real life.
