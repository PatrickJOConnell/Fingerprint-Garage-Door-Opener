This project utilizes a fingerprint scanning unit (FPS GT-521F32) to scan and identify fingerprints and open or close the garage door with verified fingerprint IDs. This process is controlled by Kuman UNO R3 Development Board(s) and programmed with open-source Arduino Software (ARDUINO 1.8.5)
