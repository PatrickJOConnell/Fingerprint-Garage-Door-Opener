Structure

The structure of the prototype was created using a smooth particle board. The panel of the garage door was created using Sheet Metal cut in 2" and connected using hinges. Wooden dowels are added to the panel to assist the panel from receding and advancing. There is a wooden roller hinged in between the smooth particle board that turn clockwise and counter-clockwise which is hooked up to the panel using Fishing Line which hence moves the panel of the garage. As the motors receive signal from the arduino that a matched fingerprint has been encountered. It will perform open and close on the panel of the garage based on the current status of the garage.

Circuit Compartment

A compartment has been created for the circuit to live in. The main reason for this is to house the circuit in without getting in the way and/or allow still allow full functionality to the motors of the structure. The compartmatilization of the circuit allows the us to fully control the circuit. For which, we have screwed it on top of the roof of the strucure. Which allows the circuit to be there safely also it does not move when we try to carry the structure around.
